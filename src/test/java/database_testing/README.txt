1.Download database drivers & associate to your project from the below link.

https://www.oracle.com/database/technologies/jdbcdriver-ucp-downloads.html

I am using oracle 11g, hence am downloading the related ojdbc.jar file[Goto the above link]

Copy & paste the ojdbc.jar file in ur drivers folder & configure the build path.

Steps to connect database & execute queries through java program.

1.We need to establish database connection through java program using 'Connection' class
2.We need to create a statement[i.e Query] within the eclipse itself by using 'Statement' class
3.We need to execute this statement[Query] using Statement object
4.we need to close the connection
